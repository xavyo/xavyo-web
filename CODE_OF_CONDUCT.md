# Code of Conduct

## Our Standards

This project maintains a professional environment focused on technical excellence.

**Expected behavior:**
- Constructive technical discussions
- Respectful disagreement on technical matters
- Helpful code reviews and feedback
- Clear and professional communication

**Unacceptable behavior:**
- Personal attacks or insults
- Harassment of any kind
- Spam or off-topic disruption
- Sharing others' private information

## Scope

This applies to all project spaces: issues, pull requests, discussions, and related communication channels.

## Enforcement

Violations may result in:
1. Warning
2. Temporary ban
3. Permanent ban

Report issues to: pascal@heartbit.ai

## Focus

We're here to build great software. Keep discussions technical, stay professional, and help each other succeed.
